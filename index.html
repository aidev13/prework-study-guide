Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git pull origin main: Pulls the latest 'main' branch.
  • git add -a: Commits all files in the working branch
  • git commit -m: Use the '-m' flag to associate a message with our commit. Commit messages should be short descriptive messages of the changes implemented.
  • git push origin: Pushes changes in the origin to the repo
  • "Issues" in github = tasks to do, or task management.
